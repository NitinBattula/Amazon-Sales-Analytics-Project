# Amazon Sales Data Analysis

This project demonstrates end-to-end data analysis using Python, SQL, and Power BI.

## Tools Used
- Python (Pandas)
- SQL
- Power BI

## Workflow
- Cleaned raw Amazon sales data using Pandas
- Created a cleaned dataset (sales_cleaned.csv)
- Analyzed data using SQL queries
- Built a Power BI dashboard for visualization

## Folder Structure
- data/raw – Original dataset
- data/cleaned – Cleaned dataset
- notebooks – Data cleaning notebook
- sql – SQL queries for insights
- powerbi – Dashboard screenshot
